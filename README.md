# Calculator

A simple, interactive calculator built with HTML, CSS, and JavaScript as part of the Foundations Course of The Odin Project. This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, and application state management.

## Repository

🔗 [Source Code](https://github.com/alexybnz/odin-calculator)

## Features

- Perform basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (÷)
- Chained calculations
- Clear functionality
- Decimal support
- Error handling for division by zero
- Responsive design for different screen sizes
- Keyboard support *(if implemented)*

## Built With

- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned

Through this project, I practiced and improved my understanding of:

- JavaScript functions and scope
- DOM manipulation
- Event listeners
- Managing application state
- Conditional logic
- Debugging complex interactions
- Creating responsive layouts with CSS

## Challenges

One of the biggest challenges was managing the calculator's state while handling chained operations and edge cases, such as:

- Preventing multiple operators from being entered consecutively
- Handling decimal inputs correctly
- Clearing and resetting values appropriately
- Preventing invalid calculations like division by zero
